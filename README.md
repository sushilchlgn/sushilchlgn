## About

Most of my work has gone into a notification pipeline for college notices, where I’ve explored matching engines, scope hierarchies, OAuth microservices, and real-time delivery flows. Outside of building systems, I spend a good amount of time trying to break them — especially around bug hunting and application security.

## Featured Projects

### Notice Alert System
A pipeline that scrapes college notice boards, matches new notices against a scope hierarchy, and pushes alerts as soon as they appear. The system uses Firebase Cloud Messaging and Go-based microservices, with a React Native prototype for mobile delivery.

### Ward-Level Civic Alerts
A concept for delivering local government notices directly to residents over SMS or Telegram, avoiding social-media feed ranking and making civic updates easier to access.

### Security Testing & Bug Hunting
Hands-on practice around OAuth flows, JWT validation edge cases, and static analysis of Go services. I also work through security labs and bug bounty-style exercises to sharpen practical testing skills.
